3.存储结构
innodb是默认开启autocommit的，就是每条语句都提交，效率低。可以把语句封在begin和end之间批量执行。
myisam 是 ISAM（Indexed Sequential Access Method：有索引的顺序访问方法）所改良，性能好，但不支持事务
